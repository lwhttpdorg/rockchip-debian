# Enabled Interface Pins on Radxa Zero 3W

## I2C

| Interface | Pin | GPIO | Function |
|------|------|------|------|
| I2C4 | Pin 27 | GPIO4_B2 | SDA |
| I2C4 | Pin 28 | GPIO4_B3 | SCL |

| Interface | Pin | GPIO | Function |
|------|------|------|------|
| csi_dphy | - | - | MIPI CSI-2 PHY |
| I2C2 | CSI Pin 20 | GPIO4_B5 | SCL |
| I2C2 | CSI Pin 21 | GPIO4_B4 | SDA |
